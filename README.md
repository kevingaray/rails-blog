# README
Bloggin web app created in the Ruby on rails dev course on Udemy

* Ruby version '3.1.2'
* Rails version '6.1.7'


To run this locally, create a .env file with credentials for the postgres database
```
DB_USERNAME = postgres
DB_PASSWORD = admin
```
